# InhaleToEat
Tool to gain confidence in the use of Afrezza inhalable insulin.

"InhaleToEat" is a trademark of Eitan Rosa.  
Unauthorized use of the name or branding is not permitted.

---

## Status

⚠️ Early-stage release

This project is under active development and shared for
validation, testing, and feedback.

Features, dosing logic, and UI may change as the tool evolves.
It is not intended to replace clinical judgment or professional care.

---

## Disclaimer

InhaleToEat is a decision-support tool based on user-entered data and
personal experimentation. It does not provide medical advice and is not
a substitute for professional medical guidance.

Users are responsible for verifying all dosing decisions and outcomes.

---

## Getting Started

To use InhaleToEat:
1. Open `index.html` in your browser
2. Install as a PWA (optional)
